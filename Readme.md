# FMAUnit4PYNQ  

![image](./Img/architecture_ip.png)

## Overview  

このプロジェクトはPYNQ-Z2のPL上にFMAユニットを搭載して計算するだけのお遊びプロジェクトです。

## Requirements

- TUL PYNQ-Z2 Board
- PYNQ v2.6
- Vivado 2020.1 (Linux ver.)
- Vitis HLS 2020.1 (Linux ver.)
